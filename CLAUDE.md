# Инструкции для Cline — Ultrasound Project

## Работа с Git
- НЕ выполняй git-команды без явного разрешения пользователя
- НЕ делай commit, push, pull, merge, rebase и другие git-операции самостоятельно
- Если нужно проверить изменения — читай файлы напрямую
- Если пользователь хочет выполнить git-команду, он скажет: "Выполни команду git..."

## Shell
- Проект работает на Windows
- Используй PowerShell для выполнения команд
- В PowerShell используй `;` для разделения команд (НЕ `&&`)
- Избегай команд, специфичных для bash/zsh (например, `source`, `export VAR=value`)

## Общие правила
- Следуй инструкциям из файла `.clinerules` в корне проекта
