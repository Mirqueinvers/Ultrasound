# Ultrasound Desktop

Десктопное приложение для ведения протоколов ультразвуковых исследований. Electron + React + TypeScript.

## Возможности

- **Учёт пациентов и исследований** — создание, поиск, редактирование, удаление
- **Протоколирование УЗИ** — конструктор протоколов для различных типов исследований (брюшная полость, почки, щитовидная железа, сосуды и др.)
- **Мобильная синхронизация** — подключение мобильного приложения по QR-коду для приёма данных
- **Импорт из Medison** — автоматический мониторинг XML-файлов от аппаратов УЗИ
- **Печать протоколов** — форматированный вывод на принтер
- **Статистика** — аналитика по исследованиям, пациентам, типам исследований
- **Журнал** — ежедневный/периодический просмотр записей

## Технологии

| Компонент | Технология |
|---|---|
| Фреймворк | Electron + React 19 |
| Язык | TypeScript |
| Сборка | Vite + electron-builder |
| Стили | Tailwind CSS |
| База данных | SQLite (better-sqlite3) |
| Аутентификация | bcryptjs |
| Графики | Chart.js + react-chartjs-2 |
| WebSocket | ws (mobile sync) |
| QR | qrcode |

## Быстрый старт

```bash
# Установка зависимостей
npm install

# Запуск в режиме разработки (веб)
npm run dev

# Запуск в Electron
npm run electron:dev

# Сборка production
npm run build && npm run build:electron
```

## Структура проекта

```
Desktop/
├── src/              # React-приложение (компоненты, контексты, хуки)
├── electron/         # Electron main process (IPC, database, mobile-host)
├── build/            # Иконки и ресурсы для сборки
├── public/           # Статика
├── vite.config.ts    # Конфигурация Vite
├── tailwind.config.js
└── tsconfig.*.json   # Конфиги TypeScript
```

## Сборка дистрибутива

```bash
npm run electron:build
```

Готовый установщик появится в папке `release/`.
